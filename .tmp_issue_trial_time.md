## Description
L'affichage du temps restant d'essai est incorrect (affiche 'h' au lieu de 'jours').

### Localisation
- `src/pages/app.ts` ligne ~435

### Code problématique
```javascript
const daysLeft = Math.ceil((new Date(data.user.trial_end_date) - new Date()) / (1000 * 60 * 60 * 24));
document.getElementById('trial-days').textContent = daysLeft + 'h'; // ← BUG: 'h' au lieu de jours
```

### Problème
Le calcul donne des **jours** mais l'affichage montre **'h'** (heures), ce qui est confus.
- Si `daysLeft = 2`, l'affichage sera "2h" au lieu de "2 jours"

### Solution recommandée
```javascript
const daysLeft = Math.ceil((new Date(data.user.trial_end_date) - new Date()) / (1000 * 60 * 60 * 24));
const hoursLeft = Math.ceil((new Date(data.user.trial_end_date) - new Date()) / (1000 * 60 * 60));

// Afficher heures si < 24h, sinon jours
const timeLeft = daysLeft < 1 
  ? hoursLeft + 'h' 
  : daysLeft + ' jour' + (daysLeft > 1 ? 's' : '');
document.getElementById('trial-days').textContent = timeLeft;
```
